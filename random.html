Random number guessing :

random number guessing game is very interesting game. in this game we we need to select lower number and bigger number. then system picks a number between lower and bigger number. we need to guess that number. system will you 7 chances.

steps to build game :

First we need to import random package. so that we need to go command prompt and type "pip install random" and click enter. after that system will install the random module and it is ready to use.

Programming code :

import random
ll=int(input('enter a lower limit number :'))
ul=int(input('enter a upper limit number :'))
randoml=random.randint(ll,ul)
e=0
while e<8:
     e=e+1
   user=int(input('guess a number between '))
   if user==randoml:
     print('your are guess is correct...',randoml)
     break
   elif user>randoml:
     print('you are guessed higher number...')
   elif user lessthan randoml:
     print('you are guessed lower number... ')
   if e==7:
     print('your chances are over ')
     print('\n better luck next time..')
     break
